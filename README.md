# MindMap - Interactive Idea Visualization Tool

**MindMap** is a React-based mind mapping application for organizing ideas, projects, and notes visually. Core UI components, node linking/export features and drag-and-drop functionality implmented.

## ✨ Features
- Interactive Nodes - Drag, connect, resize mind map nodes
- Visual Linking - Auto-draw connections between ideas
- Export Options - PNG/SVG download, JSON export
- Zoom & Pan - Navigate complex maps smoothly
- Responsive - Works on desktop/mobile

## 🚀 Quick Start

```bash
git clone https://github.com/amanjakunnel/MindMap.git
cd MindMap
npm install
npm start
# Opens http://localhost:3000
